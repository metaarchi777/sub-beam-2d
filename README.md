# SUB-BEAM 2D — 연속보·부분골조 해석 도구

직접강성법(Direct Stiffness Method)으로 2D 연속보와 부분골조(기둥 포함)를 해석하여
**휨모멘트도(B.M.D) · 전단력도(S.F.D) · 처짐도**를 그리고, **A4 PDF 계산 보고서**를 생성하는
브라우저 기반 도구입니다. 설치나 서버 없이 `index.html` 파일 하나로 동작합니다.

**제작: 김수나**

> 배포 후 접속 주소: `https://<GitHub아이디>.github.io/sub-beam-2d/`

## 주요 기능

- **모델**: 스팬 1~8개, 지점별 상부/하부 기둥(층고·단면 개별 입력), 좌/우 캔틸레버
- **하중 11종**: 등분포 · 집중(임의 위치/중앙) · 삼각형(좌최대/우최대/중앙최대/양단최대) ·
  삼각형 n개(등간격 연속, 작은보 배치 패턴) · 부분 등분포(a~b) · 사다리꼴(w₁→w₂, a~b) ·
  사다리꼴(2방향 슬래브형, 램프 s) — 한 부재에 여러 하중 중첩 가능, 자중 자동 가산 옵션
- **결과**: 모델도·S.F.D·B.M.D(인장측 표시)·처짐도 동기화 크로스헤어, 지점 반력·회전각,
  부재 단부력·최대값 표, 기둥 단부모멘트·기둥전단 표, 평형 검토(ΣR=Σ하중)
- **강성행렬 뷰어**: 경계조건 적용 후의 K·F·d를 자유도 라벨과 함께 표시 (교육용)
- **A4 PDF 보고서**: 해석조건 요약 + 다이어그램 + 결과표 + 가정을 담은 PDF 파일 생성
  (외부 라이브러리 없이 앱이 직접 조립), 브라우저 인쇄(Ctrl+P)용 A4 스타일도 내장

## 빠른 시작

**로컬 실행** — `index.html`을 내려받아 더블클릭하면 끝입니다. 오프라인에서도 동작합니다
(온라인이면 IBM Plex 웹폰트, 오프라인이면 시스템 한글 폰트 사용).

**GitHub Pages 배포**

1. GitHub에서 새 저장소 생성 — 이름 예: `sub-beam-2d`, **Public** (무료 요금제에서 Pages는 공개 저장소 필요)
2. 이 저장소의 파일 전체를 업로드
   - 웹에서: 저장소 첫 화면의 *uploading an existing file* 링크 → 파일·폴더 드래그 → Commit
   - 또는 CLI:
     ```bash
     git init
     git add .
     git commit -m "SUB-BEAM 2D v1.0"
     git branch -M main
     git remote add origin https://github.com/<아이디>/sub-beam-2d.git
     git push -u origin main
     ```
3. 저장소 **Settings → Pages** → *Build and deployment* 에서
   Source: **Deploy from a branch**, Branch: **main / (root)** → Save
4. 1~2분 뒤 `https://<아이디>.github.io/sub-beam-2d/` 로 접속

`index.html`이 빌드 결과물째로 커밋되어 있으므로 별도의 빌드 파이프라인 없이 그대로 서비스됩니다.


## 해석 방법과 가정

- 절점당 2자유도(처짐 v, 회전 θ)의 Euler–Bernoulli 보 요소, 요소 강성행렬
  ```
  k = EI/L³ · [  12   6L  −12   6L
                 6L  4L²  −6L  2L²
                −12  −6L   12  −6L
                 6L  2L²  −6L  4L² ]
  ```
- 모든 지점은 수직변위 구속(v=0), 골조는 비횡변위(수평변위 무시)
- 기둥은 **원단 고정**으로 절점에 회전강성 4EI/H 제공(모멘트 이월률 1/2), 기둥이 없는 지점은 핀
- 분포하중은 선형 조각 {w₁, w₂, a, b}로 정규화 — 고정단력은 3점 가우스 적분
  (피적분식이 4차 다항식이므로 **엄밀해**), 내력도 W(x)·Mw(x)는 조각별 폐형식
- 전단변형·축변형 무시, 하중은 하향(+)
- 부호: 정모멘트(+)=하부 인장(B.M.D 인장측 표시), 전단(+)=좌측부 상향 합력, 처짐 ↓=하향

## 검증

해석 엔진은 이론해 **84항목**과 대조 검증되었습니다. 대표 항목:

| 케이스 | 이론해 | 비고 |
|---|---|---|
| 단순보 등분포 | Mmax = wL²/8, δ = 5wL⁴/384EI | 반력·처짐 포함 |
| 2경간 연속보 등분포 | M_B = −wL²/8, R_B = 1.25wL | 경간 최대 9wL²/128 |
| 삼각형(우최대) | Mmax = wL²/(9√3) @ 0.577L | R = wL/6, wL/3 |
| 삼각형 2개(중앙 대칭) | M_c = wL²/16, M_{L/4} = 5wL²/96 | n=1은 wL²/12로 회귀 |
| 캔틸레버 등분포 | M_root = −wℓ²/2 | 부반력 −wℓ²/2L 확인 |
| 강기둥 극한 | M_end → −wL²/12 | 고정단 수렴 |
| 부분/사다리꼴 | 정역학·중첩해 일치 | 가우스 FEF = 수치적분(10⁻⁷) |
| 기둥 분배 | 절점 모멘트 평형, 4EI/H 비례 | 기둥전단 = (Mn+Mf)/H |
